API Reference

RESTful API for programmatic access to all computational toolkits

Authentication

curl -X POST https://api.computelab.io/v1/auth \ -H "Content-Type: application/json" \ -d '{"api_key": "YOUR_API_KEY"}'

Endpoints

POST /v1/jobs/submit

Submit a new computational job

GET /v1/jobs/{job_id}

Get job status and results

GET /v1/jobs/list

List all jobs

DELETE /v1/jobs/{job_id}

Cancel or delete a job

Request Parameters

Parameter Type Required Description
toolkit string Yes Toolkit identifier (md, structure, quantum, etc.)
config object Yes Toolkit-specific configuration parameters
input_files array No Array of input file URLs or base64 data
priority integer No Job priority (1-10, default: 5)

Example Request

{ "toolkit": "molecular_dynamics", "config": { "forcefield": "amber_ff14sb", "simulation_time": 100, "temperature": 300, "ensemble": "NPT" }, "input_files": [ { "name": "protein.pdb", "data": "base64_encoded_data..." } ], "priority": 8 }
